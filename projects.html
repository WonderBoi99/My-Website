Annual Giving Wrap Up

Capstone Project with Sponsor Company Benevity

May 2023 - August 2023

Working together with a team of three, we are creating an annual wrap-up for each employee to share on their personal social media. The back end of the project is being developed using Java Spring Boot, enabling us to make the necessary Get Requests to Benevity's platform API and gather data from the XML responses. The user interface and infographics will be designed using HTML5, CSS3, JavaScript, and Vue.js, following the Benevity Skyline styling guidelines. Additionally, we have planned to implement serverless development on AWS by utilizing Docker containers and CICD.
